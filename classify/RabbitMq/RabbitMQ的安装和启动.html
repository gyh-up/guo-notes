RabbitMQ 的安装和启动 #

rabbitmq 和 erlang 版本需要对应。具体参考官网 RabbitMQ Erlang Version Requirements

一、安装 erlang 环境 #

erlang 有2种安装方式:

1、yum安装erlang #

第一步：新增文件 /etc/yum.repos.d/rabbitmq_erlang.repo，在文件里面新增下面内容并保存：

c
[rabbitmq_erlang]
name=rabbitmq_erlang
baseurl=https://packagecloud.io/rabbitmq/erlang/el/7/$basearch
repo_gpgcheck=1
gpgcheck=1
enabled=1
# PackageCloud's repository key and RabbitMQ package signing key
gpgkey=https://packagecloud.io/rabbitmq/erlang/gpgkey
https://dl.bintray.com/rabbitmq/Keys/rabbitmq-release-signing-key.asc
sslverify=1
sslcacert=/etc/pki/tls/certs/ca-bundle.crt
metadata_expire=300
[rabbitmq_erlang-source]
name=rabbitmq_erlang-source
baseurl=https://packagecloud.io/rabbitmq/erlang/el/7/SRPMS
repo_gpgcheck=1
gpgcheck=0
enabled=1
# PackageCloud's repository key and RabbitMQ package signing key
gpgkey=https://packagecloud.io/rabbitmq/erlang/gpgkey
https://dl.bintray.com/rabbitmq/Keys/rabbitmq-release-signing-key.asc
sslverify=1
sslcacert=/etc/pki/tls/certs/ca-bundle.crt

二、安装 RabbitMQ #

1、下载 rpm 包

wget https://github.com/rabbitmq/rabbitmq-server/releases/download/v3.8.1/rabbitmq-server-3.8.1-1.el7.noarch.rpm
1

2、安装 rpm 包

yum install rabbitmq-server-3.8.1-1.el7.noarch.rpm
1

3、启动rabbitmq

rabbitmq-server start
1

加入开机启动：

chkconfig rabbitmq-server on
1

4、执行命令，查看 rabbitmq 的状态

rabbitmqctl status
1

至此，rabbitmq 安装成功 5、启用 web 管理插件，这样子才能支持浏览器访问 rabbitmq

rabbitmq-plugins enable rabbitmq_management
1

6、设置操作系统字符集 utf-8

vim /etc/profile
1

文件末尾加上:

export LC_ALL=en_US.UTF-8
1

然后执行 :

source /etc/profile
1

7、增加远程访问 web 管理台用户及授权

rabbitmqctl add_user test 123456 #增加test用户，密码为123456
rabbitmqctl set_user_tags test administrator #添加test角色为 administrator
rabbitmqctl set_permissions -p "/" test ".*" ".*" ".*" #给test用户授权
rabbitmqctl list_users #查看用户是否已添加

8、输入 ip:15672 进行访问

管理控制台可以成功访问，输入用户名密码，点击 login，进入到 rabbitmq 的浏览器管理页面

三、卸载 RabbitMQ #

bash
/sbin/service rabbitmq-server stop 
yum list | grep rabbitmq 
yum -y remove rabbitmq-server.noarch  
yum list | grep erlang 
yum -y remove erlang-* 
yum remove erlang.x86_64  
rm -rf /usr/lib64/erlang 
rm -rf /var/lib/rabbitmq
